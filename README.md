# 📊 Impact of Screen Time on Lifestyle

A data analysis project that explores the relationship between daily screen time and various lifestyle factors, including BMI, sleep duration, meal habits, and family interaction. The project uses primary survey data and applies Exploratory Data Analysis (EDA) techniques to uncover meaningful insights.

---

## 📌 Project Overview

With increasing digital device usage, understanding its impact on health and lifestyle has become increasingly important. This project analyzes survey responses to identify patterns and correlations between screen time and physical as well as behavioral health indicators.

---

## 🎯 Objectives

- Collect real-world lifestyle data through a structured survey.
- Clean and preprocess the collected dataset.
- Perform Exploratory Data Analysis (EDA).
- Identify trends and correlations between screen time and health metrics.
- Present findings using clear visualizations.

---

## 📂 Dataset

- **Source:** Primary survey conducted using Google Forms
- **Sample Size:** 50 respondents
- **Data Collected Includes:**
  - Daily mobile screen time
  - Other device screen time
  - Total screen time
  - Height
  - Weight
  - BMI
  - Sleep duration
  - Screen usage during meals
  - Family interaction during meals
  - Mental well-being rating
  - Demographic information

## 📊 Key Findings

- Participants with higher daily screen time generally exhibited higher average BMI.
- Increased screen usage was associated with reduced sleep duration.
- Screen usage during meals showed a negative relationship with family interaction.
- Data visualizations helped communicate behavioral trends effectively.
